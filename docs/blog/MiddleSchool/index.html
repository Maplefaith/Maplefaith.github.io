    我的初中生活¶

    学习Tips¶

    • 学习不需要太多的规划，重在“实干”；无意义的思考带来思维负担

    • 不同的阶段有不同的任务：

    时间段 细节
    考试前、日常学习 韬光养晦。不急不慢，把每一个知识点学扎实、弄清楚。
    考试阶段 趁热打铁。及时总结学习成果，摆好自己学识、思维的水果拼盘。
    假期 保留自己的星辰，探索自己的兴趣。（初中的我是各种出去学习的，也花了爸妈不少钱，似乎现在更卷？）

    • 时间管理：学会各类状态之间的转移，不扭捏，不左右摇摆

    • 意志力：“倒也要倒在离梦想最近的地方。”

    本次整理的经验教训

    附1：优秀学生陈述¶

    阅读提醒

    下面文字生成于八年级下。

    笔者注：似乎比较早的时候就有终身学习的观念。

    思想是一切行动的源头。我积极乐观，明辨是非，全面思考。我认为：“生命会随光阴一起流逝，我们要完成生命的使命。”这，就需要我们给人类、世界做出有意义的事情。我懂得知晓国家大事，学习理解历史，明确爱国情怀。我针对于生活各个方面，像如何做人、如何待人，如何劳逸结合，都有听取他人的教诲、评论等，形成自己的思考。

    我的思想告诉我，得不忘学习。可以肯定：“不同学科，不同方法”。但是知识总是有规律的，它们有共同的特点，我得出我的思考方式：理解知识意义，深究知识内涵，善于总结。

    学习数学新知识平行四边形之前，我先了解这一知识的应用：生活中的许多东西都可以利用这一知识，且在学习中可以锻炼我的多角度思考的能力。因此，我就会产生更加强烈学习的动力。

    学习新知识时，我上课认真听讲，思维活跃；课后多做题目并分析，借业余时间查阅书本和网络、询问老师，逐渐发现这个单元的实质：线段平行与相等的应用。所有的题目都是由这些基本的内容层层铺垫而成的，掌握这一内容，便能较为轻松地知道了解题思路，更好地学习。比如，在遇到中位线与等边三角形相结合的题目时，我运用每一个图形所具有的性质，将它们串联起来分析它的解题关键所在。

    学习完这一单元或其中的某一知识后，有必要时，我会花点时间总结：归纳题目的套路和对题目类型的记忆和理解。我会请教老师，参考学习提纲和学习方法，并有所选择，从而形成自己的思路和提纲。这就是我所一直保持着的优秀成绩的重要因素。

    我的体育运动也受到我的思想影响，在体质方面，我在体质检测中是优秀的。同时，我知道运动可以放松身心，体验生命的生机。所以，和同学们一起打球，出去一同游玩，是我的生活学习常事。

    我时刻提醒自己：名利只是生活学习中的最微小的一部分。淡泊名利，从容淡定，真诚真情做实事，相信我的未来会越来越精彩。

    附2：返校宣讲笔记（未讲过）¶

    阅读提醒

    下面文字生成于高一年级。

    笔者注：可能会有自初中以来继承的一些凡尔赛。

    我知道，不到考前几天，你们的心情从不紧张。因为我也一样，每次考试，都只有到了考前几天才心理紧张起来，知道要更严肃了。其实，我们为什么要这么早紧张起来呢？除了伤身耗力之外，就还有一个好处：你能够在有限的时间里面，做出更多趁热打铁的事情。 这类事情，往往能够巩固你的现有的知识，锻炼你的心理素质和考试素质，这必将能为你的考试做准备，而且也能丰富你整个人生的阅历。做这类事情，我必须要点明合理的方法，即能够让你成功趁热打铁。

    第一点，要多做题目。与我们所谓的“刷题”是一样的道理。题目你有时觉得太少，那你就自己找题目去做。

    在这个过程中，我要强调几点：

    第一，你要明确自己刷题的意图——就是要使自己的题库更丰富，以使你考试的时候能够更娇轻车熟路——因此，你要心里知道，对于有难度、有意思的题目，一定要记录下来：现在你认为你的脑子可以，那你就记在脑里绝对没有问题；如果你真的认为自己需要整理归纳与提炼，那就买个笔记本按你自己的格式去走——没必要那么好看，就像我们老师说的；也没必要在中考前反复地读，因为这个时候你已经牢牢得将它印在脑子里了。

    第二，在这样的意图之下，你要自己主动地去做近几年的中考试卷（也许老师已经给你做了），去刷好的题目。虽然中考试卷是人纯脑子出的，而不是从别的地方扒来的，但是这个人也是人，他符合一个人的特质，只是阅历比你丰富，所以你要去迎合出题人，去考虑出题人的角度。你们知道我这几个学期的语文期末分数吧，为什么从八年级以来我总是能在温州市期末语文试卷中获得高分？如果说自己在多次训练后的感觉是重要的一块，那么猜透他的出题套路便是另一个制胜法宝。因为我在适应这类的考试，所以我能够考出好的分数。

    第三，别害怕错题。还是那句老话：别人笑你那就笑你呗，反正你收获的是给予自己内心的。遇到错题，慌张肯定是有的，这个人嘛，无法避免；但是更多的应该是虚心地求教，这个题目我错在哪里，如果是下次，我难道还会这样错吗？找到错题的原因，并整理归纳——这里我认为你一定要记录在本子上，专门的本子上——因为我们身为一个会得意忘形的人，犯了错之后总是很快就忘了疼——所以常常拿出来看看，有助于你更好地巩固。比如说，在数学的绝对值问题里，我用了一个很麻烦的分类讨论去做，结果老师讲解的时候就运用了它的几何意义——点到点的距离——马上就解决了。这时，我就要马上趁热打铁记录下所感悟的这一切，并在每次做练习时想过一遍又一遍，那么这一数学思想就成为你自己的常常想到的东西了。

    讲了这么多，也许你会泄了点气。为什么刷个题要耗费这么多精力？但是你想，如果我不耗费这么多精力，那么我的题不就白白地被刷了一遍之后被我的那个容易遗忘的大脑抛在了时间的尘埃里而再也无法想起了吗？多用点时间精力在巩固上，比一味的刷题要高效多了。我还记得我们语文老师曾经说过：“只有你付出了，后来才会有正当的回报。”现在这句话解释在这里就尤有它的意义了。

    第二点，每周来一次灌鸡血活动。所谓灌鸡血，表面上，就是好好地跟你讲讲别人的故事，以此激励你；暗地里，就是把现在的好端端的你放到和你极不愿意相比的那个人旁边，狠狠地把你的缺点和别人的优点来个比较。就是说，别人多么的悲壮，而你现在却如此平淡；不行，我一定要让你紧张起来。

    为什么我们需要打鸡血？人的动力往往是三分钟热度，在时间的洗刷和旁边人的冷淡之下，我们的动力很快就几天时间，就会丧失一大部分。这样，也正常，因为我们无法长时间地高功率工作——但是我并没有排除我们可以常常地爆发一次。在这里，我也只能隔个几周和你们来讲几句话。这周，我就送给你们这句话：“倒也要倒在离梦想最近的地方。”

    这句话是我在夏令营的时候，从那个老师那里听过来的。当时的我就有着三分钟热度，仿佛自己什么都能干好。我现在还依稀记得一点。这是瓯海中学的一个两个同学。他们曾经在全班同学面前放言狠话，如果高考考不了全校第一，那他们就跳到校旁边的河里去。后来，他们每天就死坐在教室里，凌晨半夜起来读书。

    你们知道后来结果怎么了？

    确实，高考考第一是有难度的，没有一个人上了第一；但是他们两个都上了段里前十，有个是第二名。但是他们没有办法，必须信守承诺。全班同学找了个时间，把他们共同聚在河边，脱了衣服就跳了下去。放心，他们会游泳；当然，全班同学也都笑了一堂。

    这个故事就这样，但是我想就此告诉你们，我们需要有梦想。这个梦想还必须需要你付出相应的代价。我并不是想要让你们都去跳河，万一你们不会游泳呢？但是这个要求必须对于你自己要有高度，它会逼迫着你去奋斗，去追逐梦想。这些写下来当然只是白纸黑字，我希望我们的班主任老师能够每周一次宣读一下这样的梦想，每个同学每周都要重新再受到一次刺激。

    我在当时就写下了一定要激扬青春，考上温一中。但是，我也是验证了这句话：“倒也要倒在离梦想最近的地方。”虽然梦想没有实现，但是我倒在了离它最近的地方：龙湾中学。这只是在名分上；在真正的学识的积累上，那一个初三的学期时间是我一生初次经历的最繁忙的一段时光；所有的这些，对于我在未来高中的学习，必然是非常有帮助的。刚才说的那两个同学，也倒在了离梦想最近的地方，有一个考了全段第二，必然也是一个重点大学。这个对于他的人生也必然没有过多的遗憾。

    因此，我想说：“中考的分数就是你的最近的梦想。哪怕你倒在了它的前面，这对于你现在的自己也必然有很大的改观。”
